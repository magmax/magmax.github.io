Micro-receta: Java en Awesome WM

2013-10-02 // 2013-10-02

Comments

Miguel Ángel García

Después de buscarlo varios cientos de veces, he decidido escribirlo aquí.

La forma de ejecutar programas Java sin problemas en Awesome WM es impersonándose como otro Windows Manager:

$ wmname LG3D

El programa wmname se encuentra en en paquete debian suckless-tools.
