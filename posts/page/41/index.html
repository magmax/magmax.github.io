XML vs JSON

Últimamente estoy viendo bastantes comparativas entre JSON y XML. No quería ser menos, así que investigué un poco para poder entrar en este tipo de comparativas. Para mi desgracia, me he dado cuenta de que no son comparables. Comparar XML con JSON es como comparar un documento de OpenOffice con un archivo de texto. Son dos cosas completamente diferentes. Un archivo JSON es un conjunto de datos agrupados. Nada más.
