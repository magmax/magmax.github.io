Monitorización con Graphite

Actualmente está de moda utilizar Graphite como base de cualquier sistema de monitorización, y sobre él añadir distintas herramientas. Éstas pueden estar orientadas al sistema de almacenamiento, interfaz para programas o interfaz para humanos. En este post analizaré el stack de Graphite, proponiendo diversas herramientas cuyo uso es bastante habitual. Al final del post, cualquiera con una Debian/Ubuntu debería tener instalado un Graphite básico con un panel muy molón. Graphite Es el corazón de todo el tinglado que vamos a montar.
