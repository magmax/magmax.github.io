Rails: Crear un ToDoList básico en 1 minuto

2010-12-23 // 2010-12-23

Comments

Miguel Ángel García

Ahora me ha dado por probar Ruby On Rails. Y aquí tenéis los pasos para crear un ToDo List muy básico:

# $ rails todolist # $ cd todolist # $ ruby script/generate scaffold Task title:string description:text priority:integer effort:integer duedate:datetime scheduleddate:datetime scope:string # $ rake db:create # $ rake db:migrate # $ ruby script/server # Abrir un navegador en la ruta http://localhost:3000/tasks

He estado probando a meter referencias a otras tablas, pero el andamio (scaffold) no es muy bueno para eso.

Por defecto ya tiene configurada una BBDD sqlite que es la que estamos usando.

En general, me está gustando bastante la filosofía de Ruby On Rails, y veo que es una herramienta más depurada que Kohana o, incluso, Django (muy a mi pesar).
