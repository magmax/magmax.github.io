vnstat: a console-based network traffic monitor

Please, if you have not done it, read the Debaday article. Thank you. This article of debaday site, by “András Horváth”:https://log69.com/index_en.html, were posted at https://debaday.debian.net/2009/02/08/vnstat-a-console-based-network-traffic-monitor/ on 08/02/2009. There are many command-line utilities in the Unix / Linux world that collect network traffic statistics for a chosen interface. vnstat is different because statistics are collected permanently. It keeps the data in files so traffic monitoring is resumed at reboot. vnstat can be useful in situations when you need to know the exact amount of traffic that goes through the interface for a period longer than the computer is on.
