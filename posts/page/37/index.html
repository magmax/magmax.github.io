Apache Derby, tu base de datos relacional

Realizando los ejercicios de un curso que estoy haciendo, me he visto obligado a usar una base de datos desde Java. Había oído hablar de “derby”, pero no tenía muy claro su uso ni ventajas. Aunque aún estoy en ello, creo que puedo ver claramente estas ventajas, así que voy a contar cómo usarla. Ventajas Las grandes ventajas de Derby son: Es una base de datos nativa en java El núcleo y el driver ocupan menos de 3Mb Aunque tiene modo cliente-servidor, se pude embeber en nuestra aplicación.
